# Brand Personality

The brand should feel:

- Technical
- Structured
- Calm
- Modern
- Minimal
- Precise
- Intentional
- Engineering-oriented

The brand should NOT feel:

- Overly corporate
- Loud
- Trend-chasing
- Over-animated
- Playful startup aesthetic
- Generic portfolio template
- “AI guru” branding

---

# Communication Style

Tone of voice:

- Clear and concise
- Technical but readable
- Confident without exaggeration
- Analytical
- Product-oriented
- Minimal marketing language

Avoid:

- Excessive buzzwords
- “Revolutionary”
- “Innovative future”
- “World-class”
- “Changing the world”
- Overly motivational copy

Preferred language style:

- Short sentences
- Dense information
- Structured explanation
- Engineering vocabulary where appropriate

# Motion Principles

Animations should:

- Feel responsive
- Be subtle
- Improve hierarchy
- Improve interaction clarity

Avoid:

- Floating objects
- Heavy parallax
- Random glow effects

Preferred motion:

- Fade

- Hover elevation
- Opacity transitions
