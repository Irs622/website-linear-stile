# Irsal Shydiq Portfolio

Modern engineering-focused portfolio website built with Next.js, TypeScript, TailwindCSS, and MDX.

Designed as a long-term digital identity and technical showcase focused on software engineering, AI-assisted development, and modern product systems.

---

# Overview

This project is a personal portfolio and technical branding platform.

The website combines:

- Engineering portfolio
- Technical writing
- Project case studies
- Research documentation
- Product-oriented presentation

The overall experience is inspired by modern software products such as Linear, Vercel, and Raycast.

---

# Design Direction

The interface follows a Linear-inspired design system:

- Near-black canvas
- Minimal accent colors
- Strong typography hierarchy
- Dense but readable layout
- Product-focused visual rhythm

The website prioritizes clarity, structure, and technical presentation over decorative visuals.
