## VeriLog Audit Engine

### Category

Distributed Systems · Security Engineering

---

### Status

In Development

---

### Overview

VeriLog is a decentralized and tamper-evident audit logging system designed for distributed and microservice-based environments.

The system focuses on preserving log integrity using cryptographic verification and optional blockchain anchoring mechanisms.

---

### Problem

Traditional logging systems are vulnerable to:

- Log tampering
- Unauthorized modification
- Centralized failure points
- Lack of integrity verification

Modern distributed systems require more trustworthy audit infrastructures.

---

### Solution

VeriLog introduces:

- Cryptographic hashing
- Immutable audit chains
- Distributed verification
- Optional blockchain anchoring
- Structured event tracking

The system is designed with performance, traceability, and integrity as primary goals.

---

### Technical Focus

- Distributed systems
- Cryptographic verification
- Audit architecture
- High-performance backend engineering
- Concurrent processing

---

### Stack

- Rust
- Axum
- PostgreSQL
- Docker
- Blockchain Integration

---

### Key Features

- Immutable log chain
- Hash verification
- Tamper detection
- Distributed node support
- Secure audit trail

---

### Engineering Challenges

- Maintaining high write performance
- Designing immutable verification flows
- Balancing security and scalability
- Structuring concurrent log processing

---

### Learning Outcomes

- Distributed system architecture
- Rust memory safety
- Concurrent backend processing
- Security-oriented engineering

# SleepTrack

### Category

Human–Computer Interaction · Behavioral Technology

---

### Status

Research Concept

---

### Overview

SleepTrack is a behavioral design application focused on improving sleep quality by analyzing digital habits and screen-time behavior.

The project combines Human–Computer Interaction principles with behavioral change methodologies.

---

### Problem

Excessive digital activity and poor screen-time habits negatively affect sleep quality among students and young adults.

Most existing applications focus only on tracking sleep duration without addressing behavioral triggers.

---

### Solution

SleepTrack focuses on:

- Digital behavior monitoring
- Habit analysis
- Sleep pattern visualization
- Behavioral intervention design
- Personalized recommendations
